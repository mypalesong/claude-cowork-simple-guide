# 회의 & 협업 활용법

> 회의 전, 회의 중, 회의 후 — Claude가 모든 단계를 도와줍니다.

## 회의 전: 안건 정리

```text
내일 오전 10시 마케팅팀 주간회의 안건을 정리해줘.

논의 주제:
- 신제품 론칭 캠페인 진행 현황
- SNS 광고 성과 리뷰 (지난주 데이터)
- 다음 분기 예산 배분
- 인플루언서 협업 건 의사결정

각 안건별로 예상 소요시간도 배분해줘. 총 1시간.
회의 초대 메일에 넣을 간단한 안내문도 작성해줘.
```

## 회의 후: 회의록 정리

```text
아래 회의 내용을 회의록으로 정리해줘:

[녹음 텍스트 또는 메모를 붙여넣기]

양식:
1. 회의 정보 (일시, 참석자, 장소)
2. 주요 논의사항 (안건별로 구분)
3. 결정사항 (명확하게)
4. 액션 아이템 (담당자, 기한 포함)
5. 다음 회의 일정

액션 아이템은 표로 정리해줘:
| 번호 | 할 일 | 담당자 | 기한 | 비고 |
```

## 회의 후: 후속 조치

### 참석자 공유 메일
```text
방금 정리한 회의록을 바탕으로
참석자 전원에게 보낼 회의 결과 공유 이메일 써줘.
핵심 결정사항과 각자의 액션 아이템을 강조해줘.
```

### 불참자 요약본
```text
오늘 회의에 참석 못한 김과장님을 위해
5줄 요약본을 만들어줘.
김과장님이 알아야 할 것만 간추려서.
```

## 회의 중: 실시간 노트 활용

회의 중에도 Claude를 옆에 두고 실시간으로 활용할 수 있습니다:

```text
지금부터 회의 내용을 메모할 거야.
내가 핵심 키워드만 적어줄 테니, 나중에 회의록으로 정리해줘.
시작!

- 신제품 론칭일 4/15 확정
- 마케팅 예산 3000만→2500만 축소
- 김대리 SNS 콘텐츠 담당
- 다음 미팅 3/25 오전 10시
```

::: tip
완벽한 문장으로 적을 필요 없어요! 키워드만 던져도 Claude가 구조화된 회의록으로 정리해줍니다.
:::

## 회의 녹음 텍스트 활용

음성 녹음을 텍스트로 변환한 후 Claude에게 넘기면 더 효율적입니다:

```text
아래는 오늘 회의 녹음을 텍스트로 변환한 내용이야.
(말이 겹치거나 어색한 부분이 있을 수 있어)

[녹음 변환 텍스트 붙여넣기]

이걸로:
1. 깔끔한 회의록 작성
2. 핵심 결정사항 3가지
3. 각 참석자별 액션 아이템
4. 불명확한 부분은 "확인 필요"로 표시
```

## 팀 세션으로 협업하기

1. **팀 세션 만들기** — Cowork에서 "새 팀 세션"을 만들면 동료를 초대할 수 있습니다.
2. **함께 아이디어 발전시키기** — "우리 팀 워크숍 프로그램 아이디어를 5개 제안해줘" → 팀원들이 실시간으로 의견을 추가
3. **결과물 함께 다듬기** — Claude가 만든 초안을 팀원들이 보면서 실시간 수정 요청
