# 첫 번째 대화해보기

> 아래 예시를 그대로 따라 해보세요. 3분이면 충분합니다.

## 예제 1: 이메일 작성 부탁하기

**상황:** 거래처에 납품 일정 변경 안내 이메일을 보내야 할 때

```text
거래처 담당자(김대리)에게 이메일 써줘.
내용: 원래 3월 20일 납품 예정이었는데, 자재 수급 문제로 3월 27일로 변경.
진심으로 사과하고, 추가 비용은 우리가 부담한다고 알려줘.
톤: 정중하지만 따뜻하게
```

::: tip 포인트
Claude가 제목, 인사말, 본문, 맺음말을 포함한 완성된 이메일을 작성합니다. 마음에 안 드는 부분이 있으면 **"좀 더 격식 있게 바꿔줘"** 또는 **"마지막 문단 빼줘"** 처럼 수정을 요청하면 됩니다.
:::

## 예제 2: 파일 내용 요약 부탁하기

**상황:** 긴 회의록 파일을 빠르게 파악하고 싶을 때

```text
내 문서 폴더에 있는 "3월 전체회의록.docx" 파일을 읽고,
핵심 결정사항만 3줄로 요약해줘.
그리고 나한테 할당된 액션 아이템도 따로 뽑아줘. (내 이름: 박지연)
```

## 예제 3: 간단한 번역

**상황:** 해외 본사에서 온 영문 공지를 팀원들에게 공유해야 할 때

```text
아래 영문 공지를 자연스러운 한국어로 번역해줘.
전문 용어는 영어를 괄호 안에 남겨줘.
우리 팀 슬랙에 올릴 거라서 친근한 톤으로.

[영문 내용을 여기에 붙여넣기]
```

## 대화 이어가기 핵심

| 포인트 | 설명 |
|--------|------|
| **"이어서"가 핵심** | 같은 대화 안에서는 이전 맥락을 기억합니다. "방금 그 이메일에서 첫 문단만 바꿔줘" 처럼 이어서 말하면 됩니다. |
| **중간에 끊어도 OK** | 점심 먹고 돌아와도 이전 대화가 남아있어요. 그냥 이어서 부탁하면 됩니다. |
| **파일 첨부도 가능** | 드래그 앤 드롭으로 파일을 첨부하거나, 파일 이름을 말하면 직접 찾아줍니다. |
