# 팀 교육 프로그램

> 팀원들에게 Cowork를 가르치고 싶다면? 상황에 맞는 교육 프로그램을 골라보세요.

## 프로그램 비교

| 프로그램 | 소요 시간 | 대상 | 수용 인원 | 난이도 |
|----------|-----------|------|-----------|--------|
| ⚡ 라이트닝 세션 | 30분 | 바쁜 팀, 첫 소개 | 5~50명 | ⭐ 입문 |
| 📚 기본 교육 | 1시간 | 전 직원 도입 교육 | 10~30명 | ⭐ 초급 |
| 🎓 반나절 워크숍 | 3시간 | 부서별 심화 교육 | 8~20명 | ⭐⭐ 중급 |
| 🏆 마스터 코스 | 6시간 (1일) | AI 챔피언 양성 | 5~15명 | ⭐⭐⭐ 고급 |

::: tip 어떤 프로그램을 선택해야 할까?
- **처음 도입**하는 팀 → 라이트닝 세션으로 관심 유도 → 기본 교육으로 확대
- **빠르게 전사 도입**해야 한다면 → 기본 교육을 여러 차수 운영
- **특정 부서**가 적극 활용하고 싶다면 → 반나절 워크숍
- **내부 전문가**를 키우고 싶다면 → 마스터 코스
:::

## ⚡ 30분 라이트닝 세션

> 바쁜 팀에게 Cowork의 핵심 가치를 빠르게 전달하는 초경량 세션입니다.

**대상:** 바쁜 팀, 첫 소개, 경영진 브리핑
**인원:** 5~50명
**형태:** 발표 + 라이브 데모 + 간단 실습

### 시간표

| 시간 | 구성 | 내용 | 진행 방식 |
|------|------|------|-----------|
| 00:00~05:00 | Cowork란? | AI 비서의 개념, 할 수 있는 일 3가지 | 발표 |
| 05:00~15:00 | 라이브 데모 | 진행자가 실시간으로 Cowork 사용 시연 | 시연 |
| 15:00~25:00 | 따라하기 | 참석자 각자 실습 시나리오 수행 | 실습 |
| 25:00~30:00 | Q&A | 질문 & 다음 단계 안내 | 토론 |

### 준비물 체크리스트

- [ ] 프로젝터 또는 화면 공유 환경
- [ ] 참석자 전원 Cowork 계정 생성 완료
- [ ] 실습 시나리오 인쇄물 또는 공유 문서
- [ ] Wi-Fi 접속 정보 안내
- [ ] 피드백 설문 링크 (QR코드)

### 실습 시나리오: 이메일 작성

```text

- 지연 사유 (외부 API 연동 이슈)

```

- **1주 후 팔로업:** 실제 사용 여부 간단 설문 발송

### Part 3: 워크플로우 설계 (슬라이드 핵심)

**슬라이드 1 — 워크플로우란?**
- 단일 질문 → 연속 작업 파이프라인으로 전환
- 예: 회의록 → 요약 → 액션아이템 → 팔로업 이메일 (4단계 자동화)

**슬라이드 2 — 설계 3단계**
1. 반복 업무 찾기: "매주/매월 반복하는 작업이 뭔가요?"
2. 단계 분해: 입력 → 처리 → 출력으로 나누기
3. 프롬프트 체이닝: 각 단계를 연결하는 프롬프트 작성

**슬라이드 3 — 실습**
- 팀별로 반복 업무 1개를 골라 워크플로우로 설계
- 포스트잇으로 각 단계를 벽에 붙이기
- 팀별 발표 (2분씩)

### 평가 방법

- **실습 결과물 평가:** 3개 시나리오 완료율 + 품질 (동료 피드백)
- **워크플로우 발표:** 팀별 워크플로우 실현 가능성 평가 (진행자 + 동료)
- **가이드라인 합의 완료:** 4개 항목 모두 합의 여부 확인
- **만족도 설문:** 5점 척도, 자유 서술 포함 (목표 평균: 4.2 이상)

3. **구현 (25분):** 실제 프롬프트 작성 및 테스트
