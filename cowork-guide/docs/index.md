## Before & After

| 업무 | Before (수동) | After (Claude와 함께) |
|------|--------------|---------------------|
| 이메일 작성 | 15분 고민하며 작성 | 2분 — 상황만 말하면 완성 |
| 주간 보고서 | 1시간 — 데이터 취합 + 작성 | 15분 — 파일 주고 양식만 지정 |
| 회의록 정리 | 30분 — 메모 보며 타이핑 | 5분 — 키워드만 던지면 완성 |
| 데이터 분석 | 2시간 — 엑셀 수식 삽질 | 10분 — 말로 설명하면 분석 |
| 번역 | 30분 — 구글 번역 + 다듬기 | 3분 — 톤까지 맞춤 번역 |
