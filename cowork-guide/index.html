Cowork가 뭔가요?

한 마디로, 내 컴퓨터에서 나와 함께 일하는 AI 동료입니다.

기존 ChatGPT / 웹 AI와 뭐가 다른가요?

웹에서 쓰는 AI 챗봇은 "질문하면 답변하는" 수준입니다. Cowork는 "내 파일을 보고, 직접 작업까지 해주는" 데스크탑 동료예요.

상황 웹 AI 챗봇 Claude Cowork
"이 보고서 수정해줘" 파일을 복사-붙여넣기 해야 함 내 컴퓨터의 파일을 직접 열어서 수정
"지난번에 했던 거 이어서" 새 대화마다 처음부터 설명 이전 대화와 작업 맥락을 기억
"이 엑셀 데이터 분석해줘" 파일 업로드 용량 제한 내 컴퓨터의 파일에 직접 접근
"회의록 정리해서 공유해줘" 결과를 수동으로 복사해야 함 파일로 저장하고 팀에게 공유 가능
"우리 팀 규칙대로 작성해줘" 매번 규칙을 알려줘야 함 팀 규칙을 저장해두면 항상 반영

비유로 이해하기

💬

웹 AI = 콜센터 상담원

전화할 때마다 새로운 상담원. 매번 상황을 처음부터 설명해야 하고, 직접 뭔가 해주지는 못합니다.

💻

Cowork = 옆자리 동료

우리 팀 상황을 알고, 내 파일도 볼 수 있고, 직접 문서를 고쳐주기도 하는 똑똑한 동료.

Cowork로 할 수 있는 것들

  • ✓
    문서 작성: 이메일, 보고서, 기획서, 제안서 초안 작성 및 퇴고
  • ✓
    데이터 정리: 엑셀/CSV 데이터를 읽고 분석하고 요약
  • ✓
    회의 지원: 회의록 정리, 액션 아이템 추출, 후속 이메일 작성
  • ✓
    번역 & 요약: 긴 문서 핵심 요약, 다국어 번역
  • ✓
    자료 조사: 특정 주제에 대한 리서치와 정리
  • ✓
    반복 업무: 매주/매달 같은 형태의 보고서, 뉴스레터 등 자동화
i
안심하세요 코딩이나 프로그래밍 지식은 전혀 필요 없습니다. 평소에 동료에게 부탁하듯 말로 하면 됩니다.

처음 시작하기

5분이면 Cowork를 시작할 수 있습니다. 차근차근 따라해보세요.

설치하기

1

Claude Desktop 다운로드

claude.ai/download에서 내 컴퓨터(Windows 또는 Mac)에 맞는 버전을 다운로드합니다.

i
일반 프로그램 설치와 동일합니다. 다운로드 → 실행 → 다음 → 완료!
2

로그인하기

Claude Desktop을 실행하면 로그인 화면이 나옵니다. 회사에서 계정을 발급받았다면 해당 계정으로, 아니면 개인 계정으로 로그인하세요.

3

Cowork 모드 켜기

로그인 후 화면 상단 또는 설정에서 "Cowork" 모드를 활성화합니다. 이제 Claude가 내 컴퓨터의 파일을 보고, 직접 도와줄 수 있게 됩니다.

!
파일 접근 권한 처음 켜면 "파일 접근을 허용할까요?"라고 물어봅니다. 허용해야 Cowork 기능을 쓸 수 있어요. 걱정 마세요—허용한 폴더의 파일만 봅니다.
4

작업 폴더 선택

Claude가 접근할 폴더를 선택합니다. 보통 내 문서 폴더나, 현재 작업 중인 프로젝트 폴더를 선택하면 됩니다.

화면 구성 알아보기

💬

대화 영역

Claude와 대화하는 공간. 평소 메신저 쓰듯이 질문하거나 부탁하면 됩니다.

📁

파일 패널

현재 작업 폴더의 파일 목록. Claude가 이 파일들을 읽고 수정할 수 있습니다.

🛠️

도구 모음

파일 첨부, 스크린샷, 설정 등 추가 기능 버튼들이 모여 있습니다.

⚙️

설정

언어, 응답 스타일, 접근 폴더 등을 변경할 수 있습니다.

✓
준비 완료! 여기까지 했으면 Cowork를 사용할 준비가 끝났습니다. 다음 섹션에서 첫 대화를 시작해봅시다!

첫 번째 대화해보기

아래 예시를 그대로 따라 해보세요. 3분이면 충분합니다.

예제 1: 이메일 작성 부탁하기

상황: 거래처에 납품 일정 변경 안내 이메일을 보내야 할 때

내가 입력 
거래처 담당자(김대리)에게 이메일 써줘.
내용: 원래 3월 20일 납품 예정이었는데, 자재 수급 문제로 3월 27일로 변경.
진심으로 사과하고, 추가 비용은 우리가 부담한다고 알려줘.
톤: 정중하지만 따뜻하게
✓
Claude 응답 예시 Claude가 제목, 인사말, 본문, 맺음말을 포함한 완성된 이메일을 작성합니다. 마음에 안 드는 부분이 있으면 "좀 더 격식 있게 바꿔줘" 또는 "마지막 문단 빼줘" 처럼 수정을 요청하면 됩니다.

예제 2: 파일 내용 요약 부탁하기

상황: 긴 회의록 파일을 빠르게 파악하고 싶을 때

내가 입력 
내 문서 폴더에 있는 "3월 전체회의록.docx" 파일을 읽고,
핵심 결정사항만 3줄로 요약해줘.
그리고 나한테 할당된 액션 아이템도 따로 뽑아줘. (내 이름: 박지연)
✨
포인트 파일 이름을 알려주면 Claude가 직접 찾아서 읽습니다. "3줄로" 같이 원하는 분량을 알려주면 더 정확한 결과를 받을 수 있어요.

예제 3: 간단한 번역

상황: 해외 본사에서 온 영문 공지를 팀원들에게 공유해야 할 때

내가 입력 
아래 영문 공지를 자연스러운 한국어로 번역해줘.
전문 용어는 영어를 괄호 안에 남겨줘.
우리 팀 슬랙에 올릴 거라서 친근한 톤으로.

[영문 내용을 여기에 붙여넣기]

대화 이어가기 핵심

🔁

"이어서" 가 핵심

같은 대화 안에서는 이전 맥락을 기억합니다. "방금 그 이메일에서 첫 문단만 바꿔줘" 처럼 이어서 말하면 됩니다.

✋

중간에 끊어도 OK

점심 먹고 돌아와도 이전 대화가 남아있어요. 그냥 이어서 부탁하면 됩니다.

📎

파일 첨부도 가능

드래그 앤 드롭으로 파일을 첨부하거나, 파일 이름을 말하면 직접 찾아줍니다.

보고서 & 기획서

주간/월간 보고서

바로 써보세요 
아래 내용으로 주간 업무 보고서 작성해줘:

이번 주 한 일:
- 신규 거래처 A사 미팅 (수요일)
- 2분기 마케팅 예산안 초안 작성
- 인턴 교육 자료 만들기

다음 주 계획:
- A사 제안서 발송
- 팀 워크숍 준비
- 월말 실적 정리

이슈:
- 디자인팀 인력 부족으로 브로슈어 제작 지연

양식: 표 형태로 깔끔하게, 상사가 1분 안에 파악할 수 있게

기획서 초안

바로 써보세요 
사내 동호회 활성화 기획서를 작성해줘.

배경: 최근 직원 설문에서 "사내 문화 활동 부족" 의견 많았음
목표: 분기당 2개 이상 동호회 신설, 참여율 30% 달성
예산: 분기 500만원
대상: 전 직원 (약 200명)

포함할 내용:
1. 현황 분석
2. 목표 및 기대효과
3. 세부 실행 계획
4. 예산 내역
5. 일정표

A4 3장 분량으로.

보고용 정리

분석 결과를 표나 글로 정리해서 보고서에 바로 넣을 수 있는 형태로

실전 예시: 매출 데이터 분석

상황: 1분기 매출 데이터를 분석해서 팀장님께 보고해야 할 때

바로 써보세요 
"1분기_매출현황.xlsx" 파일을 열어서 분석해줘.

알고 싶은 것:
1. 월별 매출 합계와 성장률
2. 상위 5개 제품 (매출액 기준)
3. 지역별 매출 비중
4. 전년 동기 대비 변화

결과를 표로 정리하고, 핵심 인사이트 3가지를 한 줄씩 뽑아줘.
팀장님께 보고할 한 문단 코멘트도 작성해줘.

실전 예시: 설문 결과 정리

상황: 직원 만족도 설문 결과 엑셀을 받았을 때

바로 써보세요 
"직원만족도_설문결과.csv" 파일 분석해줘.

해줄 것:
1. 전체 만족도 평균 점수
2. 항목별 (급여, 복지, 성장기회, 문화) 점수 비교
3. 부서별 만족도 차이
4. 서술형 응답에서 자주 나오는 키워드 Top 10
5. 개선이 시급한 영역 3가지와 그 근거

경영진 보고용으로 깔끔하게 정리해줘.

실전 예시: 명단 & 리스트 정리

상황: 여기저기서 모은 참석자 명단을 하나로 합쳐야 할 때

바로 써보세요 
"세미나_신청자.xlsx"와 "추가신청자.csv" 두 파일을 합쳐줘.

해줄 것:
- 중복 제거 (이름+연락처 기준)
- 전화번호 형식 통일 (010-XXXX-XXXX)
- 소속별로 정렬
- 총 인원수 알려줘

합친 결과를 "최종_참석자명단.xlsx"로 저장해줘.

회의 & 협업 활용법

회의 전, 회의 중, 회의 후—Claude가 모든 단계를 도와줍니다.

회의 전: 준비

회의 안건 정리

바로 써보세요 
내일 오전 10시 마케팅팀 주간회의 안건을 정리해줘.

논의 주제:
- 신제품 론칭 캠페인 진행 현황
- SNS 광고 성과 리뷰 (지난주 데이터)
- 다음 분기 예산 배분
- 인플루언서 협업 건 의사결정

각 안건별로 예상 소요시간도 배분해줘. 총 1시간.
회의 초대 메일에 넣을 간단한 안내문도 작성해줘.

회의 후: 회의록 정리

회의 내용을 구조화된 회의록으로

바로 써보세요 
아래 회의 내용을 회의록으로 정리해줘:

[녹음 텍스트 또는 메모를 붙여넣기]

양식:
1. 회의 정보 (일시, 참석자, 장소)
2. 주요 논의사항 (안건별로 구분)
3. 결정사항 (명확하게)
4. 액션 아이템 (담당자, 기한 포함)
5. 다음 회의 일정

액션 아이템은 표로 정리해줘:
| 번호 | 할 일 | 담당자 | 기한 | 비고 |

팀 협업: Cowork 세션 활용

1

팀 세션 만들기

Cowork에서 "새 팀 세션"을 만들면 동료를 초대할 수 있습니다. 같은 Claude에게 함께 부탁하는 거예요.

2

함께 아이디어 발전시키기

예: "우리 팀 워크숍 프로그램 아이디어를 5개 제안해줘" → 팀원들이 실시간으로 의견을 추가하고, Claude가 종합합니다.

3

결과물 함께 다듬기

Claude가 만든 초안을 팀원들이 보면서 실시간으로 수정 요청. "2번 아이디어를 더 구체적으로" 등.

일상 업무 자동화

매일, 매주 반복하는 업무를 Claude에게 맡기면 시간이 확 줄어듭니다.

매일 하는 일

✉️

메일 답장 도우미

"오늘 받은 메일 3개에 대한 답장 초안을 각각 써줘"

예시 
아래 3개 메일에 대한 답장 초안을 각각 써줘:

메일 1: [내용 붙여넣기]
→ 요청 수락, 일정 확인 후 회신하겠다

메일 2: [내용 붙여넣기]
→ 자료를 첨부해서 보내겠다

메일 3: [내용 붙여넣기]
→ 정중하게 거절, 다른 담당자 연결
📝

업무일지 작성

"오늘 한 일을 업무일지 양식에 맞춰 정리해줘"

예시 
오늘 한 일을 업무일지로 정리해줘:

- 오전: A사 미팅, 제안서 피드백 받음
- 오후: 제안서 수정, B프로젝트 킥오프 미팅 참석
- 기타: 인턴 면접 참관

양식: 날짜 / 주요업무 / 세부내용 / 이슈사항 / 내일계획

매주 하는 일

📅

주간 보고서 자동화

이번 주 업무일지들을 모아서 주간 보고서로 합치기

예시 
이번 주(3/10~3/14) 업무일지 5개를 기반으로
주간 업무 보고서를 만들어줘.

- 형식: 우리 팀 양식대로 (지난주 보고서 참고)
- 핵심 성과 3가지 강조
- 이슈사항과 대응 방안 포함
- 다음 주 계획은 우선순위별로 정리
📊

주간 실적 정리

이번 주 매출/실적 데이터를 한눈에 보는 요약 만들기

예시 
"주간매출.xlsx"에서 이번 주 데이터를 분석해서
팀장님께 공유할 요약을 만들어줘.

포함할 내용:
- 이번 주 vs 지난 주 매출 비교
- 목표 대비 달성률
- 특이사항 (급증/급감 품목)
- 한 줄 코멘트

매달 하는 일

📆

월간 리포트 & 뉴스레터

바로 써보세요 
3월 팀 뉴스레터를 작성해줘.

소스:
- 이번 달 주간보고서 4개 (문서 폴더에 있음)
- 새로 합류한 팀원: 이수진 (마케팅팀, 3/4 입사)
- 팀 생일자: 김민수 (3/22)

포함할 섹션:
1. 이달의 하이라이트 (성과 3개)
2. 새 얼굴 소개
3. 이번 달 생일자
4. 다음 달 주요 일정
5. 한마디 (긍정적이고 동기부여되는 톤)
✓
시간 절약 효과 위 업무들을 수동으로 하면 하루에 약 1~2시간. Claude와 함께하면 15~30분이면 충분합니다. 나머지 시간은 더 중요한 일에 쓰세요!

기초 테스트 시나리오

처음 접하는 분도 바로 따라할 수 있는 테스트입니다. 순서대로 진행해보세요!

i
테스트 방법 각 시나리오를 순서대로 Claude에게 입력하고, 결과를 확인하세요. 예상 결과와 비교해보면 Cowork가 어떤 수준인지 체감할 수 있습니다.

Test 1: 기본 대화 능력

🎯 목표: Claude가 맥락을 이해하고 이어서 대화하는지 확인

1

첫 번째 메시지

입력 
나는 마케팅팀 대리야. 우리 팀은 5명이고, B2B SaaS 제품을 담당하고 있어.
2

두 번째 메시지 (맥락 기억 테스트)

입력 
우리 제품의 경쟁사 분석 보고서 목차를 잡아줘.

✅ 확인 포인트: Claude가 "B2B SaaS"라는 맥락을 반영했는가? 마케팅 관점의 분석인가?

3

세 번째 메시지 (수정 요청)

입력 
2번 섹션을 좀 더 구체적으로 나눠줘. 그리고 가격 비교 섹션도 추가해.

✅ 확인 포인트: "2번 섹션"이 무엇인지 정확히 이해했는가? 기존 구조를 유지하며 수정했는가?

Test 2: 파일 읽기 & 요약

🎯 목표: Claude가 내 파일을 제대로 읽고 분석하는지 확인

1

테스트 파일 준비

아무 문서 파일(.docx, .txt, .pdf)을 작업 폴더에 넣어두세요. 회의록, 보고서, 아무거나 OK.

2

요약 요청

입력 
"[파일명]" 파일을 읽고 다음을 알려줘:
1. 이 문서가 무엇에 대한 건지 한 줄 요약
2. 핵심 내용 3가지
3. 내가 꼭 알아야 할 것 1가지

✅ 확인 포인트: 파일을 정확히 읽었는가? 요약이 핵심을 잘 짚었는가?

Test 3: 문서 작성 품질

🎯 목표: 실제 업무에서 바로 쓸 수 있는 품질인지 확인

입력 
아래 상황으로 공식 이메일을 작성해줘:

상황: 우리 회사가 주최하는 세미나(4월 15일, 서울 코엑스)에
파트너사 임원들을 초대하는 이메일.

조건:
- 세미나 주제: "2026년 디지털 전환 전략"
- 기조연설: 우리 회사 CEO 김영수
- 참석 혜택: 네트워킹 디너, 백서 제공
- RSVP 기한: 4월 5일까지

톤: 격식 있되 딱딱하지 않게, 참석하고 싶게 만드는 느낌으로

✅ 확인 포인트:

  • 이메일 제목이 매력적인가?
  • 모든 정보(날짜, 장소, 주제 등)가 빠짐없이 들어갔는가?
  • 톤이 적절한가?
  • 실제로 보내도 될 수준인가?

Test 4: 표 & 데이터 정리

🎯 목표: 비정형 데이터를 깔끔하게 정리하는지 확인

입력 
아래 메모를 깔끔한 표로 정리해줘:

1월 매출 3200만 서울지점
2월은 서울 2800 부산 1500
3월 서울지점 3500만원 부산점 1800만원 대전 900
1월 부산 1200만 대전 800만
2월 대전 750만원

이걸로:
1. 지점별 x 월별 매출표 (단위: 만원)
2. 합계 행과 열 추가
3. 전월 대비 증감률도 표시

✅ 확인 포인트: 흩어진 데이터를 정확히 파악했는가? 표가 깔끔한가? 계산이 맞는가?

심화 테스트 시나리오

기초에 익숙해졌다면, 실전 업무에 가까운 테스트를 해보세요.

Test 5: 멀티 스텝 작업

🎯 목표: 여러 단계의 작업을 연속으로 처리하는지 확인

1

1단계: 상황 설정

입력 
다음 주 수요일에 우리 팀(영업팀 6명)의 팀빌딩 행사를 기획해야 해.
예산: 인당 5만원 / 장소: 강남역 근처 / 시간: 오후 2시~6시
조건: 아웃도어 활동 + 저녁 식사
2

2단계: 세부 계획 요청

입력 
첫 번째 안으로 진행하자.
세부 타임테이블과 예산 내역서를 만들어줘.
그리고 팀원들에게 공유할 안내 메시지(카톡용)도 써줘.
3

3단계: 변경 사항 반영

입력 
아, 한 명이 채식주의자야. 식당을 채식 메뉴 있는 곳으로 바꿔줘.
그리고 안내 메시지에 "편한 복장" 안내도 추가해.

✅ 확인 포인트: 이전 계획을 유지하면서 변경 사항만 정확히 반영했는가?

Test 6: 복잡한 문서 변환

🎯 목표: 하나의 내용을 여러 형태로 변환하는지 확인

입력 
우리 회사가 재택근무 정책을 변경합니다:
- 기존: 주 2일 재택 가능
- 변경: 주 3일 재택 가능 (수/목/금 중 택 3)
- 시행일: 4월 1일부터
- 조건: 팀장 사전 승인, 코어타임(10-16시) 온라인 필수
- 예외: 신입(입사 3개월 미만)은 주 1일만 가능

이 내용으로 아래 4가지를 만들어줘:
1. 전 직원 공지 이메일 (공식적)
2. 팀장용 Q&A 문서 (예상 질문 10개와 답변)
3. 사내 인트라넷 게시글 (친근한 톤)
4. 한 장짜리 요약 포스터 내용 (핵심만)

✅ 확인 포인트: 4가지 버전의 톤과 디테일이 각각 다른가? 핵심 정보는 모두 일관되게 포함되어 있는가?

Test 7: 비교 분석 & 의사결정 지원

🎯 목표: 복잡한 의사결정을 도와주는지 확인

입력 
사무실 이전을 검토 중이야. 후보지 3곳을 비교 분석해줘.

후보 A: 강남역 5분, 보증금 5억/월세 2000만, 150평, 주차 10대
후보 B: 선릉역 3분, 보증금 3억/월세 1500만, 120평, 주차 5대
후보 C: 판교, 보증금 2억/월세 1000만, 200평, 주차 20대

우리 회사 상황:
- 직원 45명, 향후 60명까지 증가 예상
- 외부 미팅 많음 (주 5-6회)
- 직원 70%가 서울 거주
- 예산: 월 총 비용 2500만원 이내

비교표 + 각 장단점 + 최종 추천(근거 포함)을 부탁해.

Test 8: 긴 문서 분석 & 질문

🎯 목표: 긴 문서를 읽고 정확하게 답변하는지 확인

입력 
"[긴 문서 파일명]"을 읽고 아래 질문에 답해줘:

1. 이 문서의 핵심 주장은 무엇인가?
2. 근거로 제시된 데이터는 몇 개이고, 신뢰할 수 있나?
3. 반대 의견이 있다면 어떤 논리가 가능한가?
4. 이 문서의 약점은?
5. 의사결정자에게 1분 브리핑한다면?

객관적으로 분석해줘. 문서에 없는 내용은 추측이라고 표시해줘.

질문 잘하는 법

같은 부탁이라도 어떻게 말하느냐에 따라 결과 품질이 확 달라집니다.

핵심 원칙 5가지

  1. 구체적으로 말하기

    ❌ Before "보고서 써줘"
    ✅ After "3월 마케팅 실적 보고서를 A4 2장 분량으로, 표 포함해서 써줘"

  2. 배경 설명하기

    ❌ Before "이메일 써줘"
    ✅ After "거래처와 첫 미팅 후 감사 인사 이메일 써줘. 상대방은 부장급이고 우리 제품에 관심을 보였어"

  3. 원하는 형태 알려주기

    ❌ Before "정리해줘"

실수 TOP 7

❌ 잘못된 예: "보고서 써줘"

✅ 올바른 예: "3월 영업 실적 보고서를 써줘. A4 2장, 표 포함, 전월 대비 비교, 팀장님께 보고할 거야"

해결법: 동료에게 부탁한다고 생각하세요. 동료에게도 그렇게 짧게 말하면 "뭘 쓰라고?" 라고 되물을 거예요. Claude도 마찬가지입니다.

문제: 첫 결과가 100% 만족스럽지 않으면 "AI는 못 쓰겠다"고 판단

해결법: Claude는 초안 작성기라고 생각하세요. 초안을 받고 → 피드백을 주고 → 수정을 받는 과정이 핵심입니다. 보통 2~3번 주고받으면 원하는 수준이 됩니다.

유용한 피드백 예시:

  • "전체적으로 좋은데, 3번 항목을 더 구체적으로"
  • "톤이 너무 딱딱해. 좀 더 부드럽게"
  • "길이가 너무 길어. 반으로 줄여줘"

문제: 새 대화를 시작하고 "아까 그거 이어서 해줘"라고 하면 Claude가 모릅니다.

해결법:

  • 같은 작업을 이어하려면 같은 대화 세션을 유지하세요.
  • 새 대화를 열어야 한다면, 필요한 맥락을 다시 알려주세요.
  • "지난번에 작성한 보고서(파일명: 3월_보고서.docx)를 수정해줘" 처럼 파일을 참조하면 됩니다.

문제: 주민등록번호, 비밀번호, 개인 카드번호 등을 입력

해결법:

  • 개인 식별 정보(주민번호, 카드번호 등)는 입력하지 마세요.
  • 필요한 경우 "XXX-XX-XXXXX" 처럼 마스킹 처리하세요.
  • 회사 보안 정책을 먼저 확인하세요.
  • Cowork는 허용된 폴더만 접근하므로, 민감 파일은 별도 폴더에 보관하세요.

문제: 숫자, 날짜, 고유명사 등이 잘못될 수 있음

해결법:

  • 팩트 체크: 수치, 날짜, 이름 등은 반드시 확인하세요.
  • 톤 체크: 보내기 전에 한 번 읽어보세요. 의도와 다른 뉘앙스가 있을 수 있습니다.
  • 컨텍스트 체크: Claude가 모르는 사내 사정이 있을 수 있습니다.
  • Claude에게 물어볼 수도 있어요: "이 내용에서 사실 확인이 필요한 부분을 알려줘"

문제: "그 엑셀 파일 열어봐" → Claude가 어떤 파일인지 모름

해결법:

  • 파일명을 정확히 알려주세요: "매출_2026Q1.xlsx"
  • 잘 모르면: "내 문서 폴더에서 '매출' 이 들어간 엑셀 파일 찾아줘"
  • 드래그 앤 드롭으로 파일을 직접 넣는 것도 좋은 방법!

문제: 10가지 작업을 한 번에 요청 → 결과 품질 저하

해결법:

  • 한 번에 3~5가지까지가 적당합니다.
  • 큰 작업은 단계별로 나눠서 요청하세요.
  • 예: "일단 목차부터 잡아줘" → (확인) → "이제 1장을 써줘" → (확인) → ...
  • 각 단계의 결과를 확인하고 다음으로 넘어가는 게 품질이 훨씬 좋습니다.

자주 묻는 질문과 답변을 모았습니다.

가장 큰 차이는 내 컴퓨터 파일에 접근할 수 있다는 점입니다. 웹 AI에게는 파일을 복사-붙여넣기 해야 하지만, Cowork는 파일명만 말하면 직접 읽고, 수정하고, 새 파일을 만들어줍니다. 또한 대화 맥락이 더 오래 유지됩니다.

회사에서 Enterprise 또는 Team Plan을 사용 중이라면, 데이터가 AI 학습에 사용되지 않습니다. 다만 대화 내용은 Anthropic 서버를 통해 처리되므로, 회사 보안 정책을 반드시 확인하세요. 주민번호, 비밀번호 같은 민감 정보는 입력하지 않는 것이 좋습니다.

Claude Desktop 자체는 무료로 설치할 수 있고, 기본 대화 기능도 무료입니다. 하지만 Cowork의 고급 기능(파일 접근, 도구 사용 등)은 Pro 이상의 유료 플랜이 필요합니다. 회사에서 도입한 경우 별도 비용 없이 사용할 수 있습니다.

한국어 매우 잘 됩니다! 비즈니스 이메일, 보고서, 기획서 등 업무 문서를 자연스러운 한국어로 작성합니다. 존댓말/반말, 격식체/비격식체도 구분해서 작성할 수 있어요. 한국어와 영어를 섞어서 사용하는 것도 가능합니다.

네, 사용해도 됩니다. 다만 몇 가지 주의사항이 있어요:

Claude의 결과물은 "아주 잘 쓴 초안"이라고 생각하고, 최종 검토는 본인이 하는 습관을 들이세요.

- 예산: 3,000만원

4. 예산 내역서
