# 처음 시작하기

> 5분이면 Cowork를 시작할 수 있습니다. 차근차근 따라해보세요.

## 1. Claude Desktop 다운로드

[claude.ai/download](https://claude.ai/download)에서 내 컴퓨터(Windows 또는 Mac)에 맞는 버전을 다운로드합니다.

::: info
일반 프로그램 설치와 동일합니다. 다운로드 → 실행 → 다음 → 완료!
:::

## 2. 로그인하기

Claude Desktop을 실행하면 로그인 화면이 나옵니다. 회사에서 계정을 발급받았다면 해당 계정으로, 아니면 개인 계정으로 로그인하세요.

## 3. Cowork 모드 켜기

로그인 후 화면 상단 또는 설정에서 **"Cowork"** 모드를 활성화합니다. 이제 Claude가 내 컴퓨터의 파일을 보고, 직접 도와줄 수 있게 됩니다.

::: warning 파일 접근 권한
처음 켜면 "파일 접근을 허용할까요?"라고 물어봅니다. **허용**해야 Cowork 기능을 쓸 수 있어요. 걱정 마세요 — 허용한 폴더의 파일만 봅니다.
:::

## 4. 작업 폴더 선택

Claude가 접근할 폴더를 선택합니다. 보통 **내 문서** 폴더나, 현재 작업 중인 **프로젝트 폴더**를 선택하면 됩니다.

## 화면 구성 알아보기

| 영역 | 설명 |
|------|------|
| **대화 영역** | Claude와 대화하는 공간. 평소 메신저 쓰듯이 질문하거나 부탁하면 됩니다. |
| **파일 패널** | 현재 작업 폴더의 파일 목록. Claude가 이 파일들을 읽고 수정할 수 있습니다. |
| **도구 모음** | 파일 첨부, 스크린샷, 설정 등 추가 기능 버튼들이 모여 있습니다. |
| **설정** | 언어, 응답 스타일, 접근 폴더 등을 변경할 수 있습니다. |

## 5. Custom Instructions 설정하기 (선택)

처음부터 설정하면 편리합니다. 설정 → Custom Instructions에서 아래 내용을 입력해보세요:

```text
- 나는 [이름]이고 [회사명] [부서]에서 일합니다
- 보고서는 존댓말, 표 포함, 간결하게 작성해주세요
- 이메일은 항상 제목을 함께 제안해주세요
- 한국어로 답변해주세요
```

::: info
나중에 언제든 수정할 수 있으니 부담 없이 시작하세요!
:::

여기까지 했으면 Cowork를 사용할 준비가 끝났습니다. 다음 섹션에서 첫 대화를 시작해봅시다!
:::
