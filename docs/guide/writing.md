# 문서 작성 활용법

> 가장 많이 쓰게 될 기능! 각 상황별로 바로 쓸 수 있는 프롬프트를 준비했습니다.

## 비즈니스 이메일

### 요청 이메일
```text
아래 상황으로 이메일 써줘:
- 받는 사람: [이름/직책]
- 목적: [구체적인 요청 사항]
- 배경: [왜 이 요청을 하는지 간단히]
- 기한: [언제까지 필요한지]
- 톤: 정중하지만 분명하게
이메일 제목도 같이 제안해줘.
```

### 사과 이메일
```text
사과 이메일 써줘:
- 받는 사람: [거래처/고객/동료]
- 무슨 일이 있었는지: [실수나 문제 상황 설명]
- 원인: [왜 이런 일이 생겼는지]
- 해결 방안: [어떻게 보상/해결할 건지]
- 재발 방지: [앞으로 어떻게 할 건지]
진심 어린 사과가 느껴지되, 비굴하지 않게.
```

### 정중한 거절
```text
정중하게 거절하는 이메일 써줘:
- 상대방 요청: [무엇을 요청받았는지]
- 거절 이유: [사실대로 or 부드럽게]
- 대안 제시: [가능하다면 다른 방법 제안]
- 관계는 유지하고 싶다는 뉘앙스로
```

## 보고서 & 기획서

### 주간/월간 보고서
```text
아래 내용으로 주간 업무 보고서 작성해줘:

이번 주 한 일:
- 신규 거래처 A사 미팅 (수요일)
- 2분기 마케팅 예산안 초안 작성
- 인턴 교육 자료 만들기

다음 주 계획:
- A사 제안서 발송
- 팀 워크숍 준비

이슈:
- 디자인팀 인력 부족으로 브로슈어 제작 지연

양식: 표 형태로 깔끔하게, 상사가 1분 안에 파악할 수 있게
```

### 기획서 초안
```text
사내 동호회 활성화 기획서를 작성해줘.

배경: 최근 직원 설문에서 "사내 문화 활동 부족" 의견 많았음
목표: 분기당 2개 이상 동호회 신설, 참여율 30% 달성
예산: 분기 500만원
대상: 전 직원 (약 200명)

포함할 내용: 현황 분석 / 목표 및 기대효과 / 세부 실행 계획 / 예산 내역 / 일정표
A4 3장 분량으로.
```

## 문서 퇴고 & 다듬기

```text
내가 쓴 글을 다듬어줘:

[여기에 내가 쓴 글 붙여넣기]

다음을 개선해줘:
- 문장이 너무 길면 나눠줘
- 어색한 표현 자연스럽게
- 오타/맞춤법 수정
- 논리 흐름이 자연스러운지 확인
원래 뜻은 바꾸지 말고, 어디를 왜 고쳤는지 알려줘.
```

::: tip 꿀팁: 톤을 지정하면 더 정확해요
"격식체로" / "반말로" / "친근하게" / "딱딱한 공문 스타일로" / "MZ세대 감성으로" — 이렇게 톤을 알려주면 Claude가 맞춤 작성합니다.
:::
