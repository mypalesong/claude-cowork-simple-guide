# 일상 업무 자동화

> 매일, 매주 반복하는 업무를 Claude에게 맡기면 시간이 확 줄어듭니다.

## 매일 하는 일

### 메일 답장 도우미
```text
아래 3개 메일에 대한 답장 초안을 각각 써줘:

메일 1: [내용 붙여넣기]
→ 요청 수락, 일정 확인 후 회신하겠다

메일 2: [내용 붙여넣기]
→ 자료를 첨부해서 보내겠다

메일 3: [내용 붙여넣기]
→ 정중하게 거절, 다른 담당자 연결
```

### 업무일지 작성
```text
오늘 한 일을 업무일지로 정리해줘:

- 오전: A사 미팅, 제안서 피드백 받음
- 오후: 제안서 수정, B프로젝트 킥오프 미팅 참석
- 기타: 인턴 면접 참관

양식: 날짜 / 주요업무 / 세부내용 / 이슈사항 / 내일계획
```

## 매주 하는 일

### 주간 보고서 자동화
```text
이번 주(3/10~3/14) 업무일지 5개를 기반으로
주간 업무 보고서를 만들어줘.

- 형식: 우리 팀 양식대로 (지난주 보고서 참고)
- 핵심 성과 3가지 강조
- 이슈사항과 대응 방안 포함
- 다음 주 계획은 우선순위별로 정리
```

### 주간 실적 정리
```text
"주간매출.xlsx"에서 이번 주 데이터를 분석해서
팀장님께 공유할 요약을 만들어줘.

포함할 내용:
- 이번 주 vs 지난 주 매출 비교
- 목표 대비 달성률
- 특이사항 (급증/급감 품목)
- 한 줄 코멘트
```

## 매달 하는 일

### 월간 뉴스레터
```text
3월 팀 뉴스레터를 작성해줘.

소스:
- 이번 달 주간보고서 4개 (문서 폴더에 있음)
- 새로 합류한 팀원: 이수진 (마케팅팀, 3/4 입사)
- 팀 생일자: 김민수 (3/22)

포함할 섹션:
1. 이달의 하이라이트 (성과 3개)
2. 새 얼굴 소개
3. 이번 달 생일자
4. 다음 달 주요 일정
5. 한마디 (긍정적이고 동기부여되는 톤)
```

::: tip 시간 절약 효과
위 업무들을 수동으로 하면 하루에 약 1~2시간. Claude와 함께하면 **15~30분**이면 충분합니다.
:::
