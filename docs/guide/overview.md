# Overview

> Claude Desktop Cowork는 이메일, 보고서, 데이터 정리, 회의록 등 일상 업무를 AI 동료와 함께 처리하는 데스크탑 협업 도구입니다. 코딩 지식은 전혀 필요 없습니다.

## 이 가이드에서 다루는 내용

| 섹션 | 설명 |
|------|------|
| **시작하기** | Cowork 개념, 설치, 첫 대화 |
| **활용법** | 문서 작성, 데이터 분석, 회의, 업무 자동화, 부서별 사례 |
| **테스트** | 직접 따라해볼 수 있는 기초/심화 시나리오 |
| **꿀팁** | 질문 잘하는 법, 자주 하는 실수, FAQ |
| **리소스** | 공식 문서, GitHub, 블로그, 영상 모음 |

## 읽는 순서 추천

::: tip
처음이라면 **Cowork가 뭔가요?** → **처음 시작하기** → **첫 번째 대화해보기** 순서로 읽고, 이후 관심 있는 활용법을 골라 보세요!
:::
