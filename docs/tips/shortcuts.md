# 단축키 & 빠른 팁

> 더 빠르게 쓰는 법! 단축키와 숨은 팁을 모았습니다.

## Claude Desktop 단축키

### 기본 단축키

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl/Cmd + N` | 새 대화 | 새로운 대화를 시작합니다 |
| `Ctrl/Cmd + K` | 빠른 검색 | 이전 대화를 빠르게 찾습니다 |
| `Ctrl/Cmd + Shift + N` | 새 프로젝트 | 새 프로젝트를 생성합니다 |
| `Ctrl/Cmd + Enter` | 메시지 전송 | 작성 중인 메시지를 전송합니다 |
| `Ctrl/Cmd + Shift + C` | 코드 블록 복사 | 코드 블록을 클립보드에 복사합니다 |
| `Escape` | 생성 중지 | 응답 생성을 즉시 중단합니다 |
| `Ctrl/Cmd + /` | 단축키 도움말 | 사용 가능한 단축키 목록을 표시합니다 |

### Cowork 전용 단축키

| 단축키 | 기능 |
|--------|------|
| `Tab` | 자동완성 수락 |
| `Ctrl/Cmd + Z` | 마지막 작업 되돌리기 |
| `Ctrl/Cmd + D` | 파일 패널 토글 |

## 숨은 기능 & 팁 10가지

1. **슬래시 명령어** — `/`로 시작하면 빠른 명령어 사용 가능
2. **드래그 앤 드롭** — 파일을 대화창에 끌어다 놓으면 자동 첨부
3. **멀티 파일 선택** — Shift 클릭으로 여러 파일 한 번에 선택
4. **대화 분기** — 이전 메시지에서 다른 방향으로 대화 분기 가능
5. **핀 고정** — 중요한 대화를 핀으로 고정해서 빠르게 접근
6. **스크린샷 붙여넣기** — `Ctrl+V`로 스크린샷 바로 붙여넣기
7. **마크다운 지원** — 입력에 마크다운 사용 가능 (`**`, `##`, `-` 등)
8. **대화 내보내기** — 대화 내용을 파일로 저장 가능
9. **다크/라이트 모드** — 시스템 설정 따라가거나 수동 전환
10. **음성 입력** — 마이크 버튼으로 음성으로 입력 가능

## 프로 팁: 시간 절약 테크닉

### "이어서" 테크닉

이전 작업을 이어갈 때 전체를 다시 설명하지 말고:

```text
방금 그 이메일에서 두 번째 문단만 더 부드럽게 바꿔줘
```

### "역할 부여" 테크닉

```text
너는 10년차 마케팅 전문가야. 이 관점에서 우리 캠페인을 평가해줘.
```

### "단계적 접근" 테크닉

```text
먼저 목차부터 잡아줘 → (확인 후) → 1장을 써줘 → (확인 후) → 2장을 써줘
```

### "비교 요청" 테크닉

```text
3가지 버전으로 써줘: 격식체, 친근체, MZ감성
```

### "검증 요청" 테크닉

```text
이 내용에서 사실 확인이 필요한 부분을 알려줘
```
