접근성 & 다양한 사용자 가이드 ​

모든 팀원이 편하게 쓸 수 있도록. 다양한 상황의 사용자를 위한 활용 팁입니다.

시니어 직원을 위한 가이드 ​

AI 도구가 처음이신 분들도 걱정하지 마세요. 복잡한 기술 지식 없이도 바로 시작할 수 있습니다.

초간단 시작법 ​

  1. Claude Desktop을 엽니다 — 프로그램 아이콘을 더블클릭하세요
  2. 궁금한 것을 그냥 물어봅니다 — 말하듯이 입력하면 됩니다
  3. 답변을 확인하고 활용합니다 — 복사해서 필요한 곳에 붙여넣기

핵심 3가지만 기억하세요

  • 질문하기: 대화창에 궁금한 것을 입력하고 Enter
  • 수정 요청: "좀 더 짧게 해줘", "존댓말로 바꿔줘"처럼 추가 요청
  • 복사하기: 답변 옆의 복사 버튼(📋)을 눌러서 원하는 곳에 붙여넣기

큰 글씨 설정 방법 ​

방법설정 경로
브라우저 확대Ctrl + + (확대) / Ctrl + - (축소) / Ctrl + 0 (원래 크기)
Windows 디스플레이설정 → 디스플레이 → 배율 및 레이아웃 → 125% 또는 150%
Mac 디스플레이시스템 설정 → 디스플레이 → 텍스트 크기 조절

자주 쓰는 프롬프트 즐겨찾기 활용 ​

매번 같은 질문을 다시 타이핑할 필요 없습니다. 자주 쓰는 프롬프트를 저장해두세요.

  • 메모장이나 스티커 메모에 자주 쓰는 문장을 저장
  • 필요할 때 복사해서 붙여넣기
  • 예시: "이 이메일을 정중하게 다듬어줘", "아래 내용을 표로 정리해줘"

TIP

처음에는 간단한 질문부터 시작하세요. "오늘 날씨 어때?" 같은 가벼운 대화도 괜찮습니다. 점점 익숙해지면 업무에 활용해보세요.

비개발 직군을 위한 가이드 ​

"코딩"이라는 단어가 나오면 당황하지 마세요 ​

Cowork는 코딩 도구가 아닙니다. 대화형 업무 도우미입니다. 기획, 마케팅, 인사, 총무, 영업 등 어떤 직군이든 업무에 바로 활용할 수 있습니다.

기술 용어 없이 설명하는 핵심 기능 5가지 ​

기능쉽게 말하면활용 예시
문서 작성초안을 대신 써줌"거래처에 보낼 감사 이메일 써줘"
문서 요약긴 글을 짧게 정리"이 보고서 핵심만 3줄로 요약해줘"
표 정리데이터를 깔끔한 표로"이 숫자들 표로 만들어줘"
번역외국어를 한국어로, 한국어를 외국어로"이 영어 이메일 한국어로 번역해줘"
아이디어 브레인스토밍함께 아이디어를 내줌"신제품 마케팅 아이디어 5개 제안해줘"

"그냥 동료에게 부탁하듯이" 말하면 됩니다 ​

이렇게 말해보세요

  • ❌ 어렵게: "JSON 형식으로 구조화된 데이터를 파싱하여..."

  • ✅ 쉽게: "이 데이터 보기 좋게 정리해줘"

  • ❌ 어렵게: "마크다운 문법을 활용한 문서 포맷팅..."

  • ✅ 쉽게: "이거 제목 넣고 번호 매겨서 깔끔하게 해줘"

외국어 화자를 위한 가이드 ​

한국어가 모국어가 아닌 팀원을 위한 팁 ​

Claude는 다국어를 지원합니다. 한국어가 어려우면 편한 언어로 시작해도 됩니다.

영어로 질문해도 OK ​

영어(또는 다른 언어)로 질문하면서 한국어 답변을 받을 수 있습니다.

text
Please summarize this document in Korean.
text
Translate the following into Korean formal style:
"We would like to schedule a meeting next week."

다국어 혼합 프롬프트 예시 ​

상황프롬프트 예시
영어→한국어 보고서"Write a summary of this report in Korean (존댓말)"
일본어 이메일 번역"この メールを韓国語に翻訳してください"
중국어 자료 요약"把这份文件用韩语总结一下"
혼합 질문"이 English document를 요약해줘"

번역 기능 적극 활용 ​

  • 회의록을 다국어로 동시 번역
  • 해외 거래처 이메일을 자연스러운 한국어로
  • 한국어 보고서를 영어로 변환하여 본사 공유
  • 전문 용어가 포함된 기술 문서 번역

TIP

Claude에게 "쉬운 한국어로 설명해줘" 또는 "Please explain in simple Korean"이라고 요청하면 외국어 화자도 이해하기 쉬운 답변을 받을 수 있습니다.

시각적 도움이 필요한 분 ​

스크린 리더와 함께 사용하기 ​

Claude Desktop은 텍스트 기반 인터페이스이므로 스크린 리더와 호환됩니다.

  • Windows: NVDA, JAWS 등의 스크린 리더를 켠 상태로 사용 가능
  • Mac: VoiceOver(Cmd + F5)를 활성화하여 사용

키보드만으로 Cowork 사용하기 ​

단축키기능
Tab다음 요소로 이동
Shift + Tab이전 요소로 이동
Enter메시지 전송 / 버튼 실행
Ctrl + C선택 영역 복사
Ctrl + V붙여넣기
Ctrl + A전체 선택
Ctrl + + / -화면 확대 / 축소

고대비 모드 설정 ​

운영체제설정 방법
Windows설정 → 접근성 → 대비 테마 → 높은 대비 선택
Mac시스템 설정 → 손쉬운 사용 → 디스플레이 → 대비 증가

음성 입력 활용 ​

타이핑이 불편하다면 음성 입력을 사용해보세요.

  • Windows: Win + H로 음성 입력 활성화
  • Mac: Fn + Fn (두 번 누르기)으로 받아쓰기 시작
  • 말로 질문하고, Claude의 답변은 스크린 리더로 듣기

INFO

음성 입력과 스크린 리더를 조합하면 키보드나 마우스 없이도 Claude를 활용할 수 있습니다.

재택/원격 근무자 ​

혼자 일할 때 Claude를 동료처럼 활용 ​

사무실에 동료가 없을 때, Claude가 대신할 수 있는 역할:

  • 검토자: "이 보고서 어색한 부분 있어?" — 제출 전 검토
  • 브레인스토밍 파트너: "이 주제로 아이디어 좀 내줘" — 혼자 고민하지 않기
  • 편집자: "이 문장 더 자연스럽게 다듬어줘" — 문서 퇴고
  • 정보 정리: "이 회의 내용 정리해줘" — 회의록 작성

시간대가 다른 팀원과 비동기 협업 ​

상황Claude 활용법
해외 팀과 협업영어 이메일 초안 작성 후 검토 요청
야간 긴급 요청Claude에게 먼저 초안을 받아 다음 날 팀원에게 전달
다른 시간대 회의 준비회의 안건과 요약본을 미리 작성
인수인계 문서현재 진행 상황을 정리하여 다음 담당자에게 공유

재택에서 특히 유용한 기능 5가지 ​

  1. 회의록 정리 — 화상 회의 내용을 붙여넣으면 깔끔하게 정리
  2. 이메일 톤 조절 — 상황에 맞는 적절한 톤으로 이메일 작성
  3. 일일 업무 보고 — 오늘 한 일을 나열하면 보고서 형식으로 변환
  4. 자료 조사 요약 — 긴 자료를 핵심만 추려서 정리
  5. 일정 정리 — 이번 주 할 일을 우선순위별로 정리

TIP

재택 근무 시작할 때 "오늘 할 일 목록 정리해줘"로 하루를 시작하고, 마무리할 때 "오늘 한 일 요약해줘"로 마감하면 업무 관리가 훨씬 수월합니다.

관리자: 팀 접근성 체크리스트 ​

팀 전체가 Cowork를 원활하게 사용할 수 있도록, 아래 항목을 정기적으로 점검하세요.

항목확인
모든 팀원이 계정을 가지고 있는가?☐
교육이 다양한 수준에 맞춰 제공되는가?☐
도움을 요청할 채널(슬랙, 팀즈 등)이 있는가?☐
다양한 기기(Mac/Windows)에서 테스트했는가?☐
피드백 수집 채널이 있는가?☐

관리자 팁

체크리스트를 분기마다 한 번씩 점검하세요. 신규 입사자가 있을 때는 반드시 온보딩 과정에 Cowork 교육을 포함하세요.

자주 듣는 걱정과 답변 ​

걱정답변
"나는 디지털에 약해서..."스마트폰으로 메시지 보낼 수 있다면 Claude도 쓸 수 있습니다. 대화하듯 글을 입력하면 됩니다.
"실수하면 어떡하죠?"실수해도 전혀 문제없습니다. Claude는 기계라서 기분 나빠하지 않고, 언제든 다시 질문할 수 있습니다. 회사 시스템에 영향을 주는 작업은 항상 확인 과정을 거칩니다.
"내 일을 빼앗기는 건 아닌가요?"Claude는 도구입니다. 계산기가 회계사를 대체하지 않듯이, Claude는 반복적인 작업을 도와서 더 중요한 일에 집중할 시간을 만들어줍니다.
"보안이 걱정돼요"회사에서 승인한 환경에서 사용하며, 민감한 정보(개인정보, 비밀번호 등)는 입력하지 않으면 됩니다. 보안 가이드라인을 따르면 안전합니다.
"나이가 많아서 못 배울까봐..."나이는 전혀 상관없습니다. Claude는 어떤 속도로 질문해도 인내심 있게 답변합니다. 오히려 풍부한 업무 경험이 더 좋은 질문을 만듭니다.

TIP

모든 사람이 AI의 혜택을 누릴 자격이 있습니다. 속도나 수준에 관계없이, 자신에게 맞는 방식으로 천천히 시작하세요. 어려울 때는 언제든 동료나 관리자에게 도움을 요청하세요.
