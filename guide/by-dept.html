부서별 활용 사례 ​

우리 부서에서는 어떻게 쓸 수 있을까? 부서별 맞춤 시나리오를 정리했습니다.

인사/HR ​

채용 공고 작성 ​

text
프론트엔드 개발자 채용 공고를 써줘.
- 경력: 3-5년 / 필수: React, TypeScript / 우대: Next.js
- 우리 회사 분위기: 수평적, 자율 출퇴근
- 톤: 딱딱한 공고 말고, 지원하고 싶게 만드는 느낌
- 연봉 범위: 5,000~7,000만원

면접 질문 준비 ​

text
마케팅팀 대리급 면접 질문을 만들어줘.
- 직무: B2B SaaS 마케팅
- 평가 영역: 직무 역량 / 문제해결력 / 협업 / 문화적합성
- 각 영역별 3개씩, 총 12개 질문
- 각 질문별 "좋은 답변 예시"와 "평가 포인트"도 같이

인사 평가 피드백 ​

text
팀원 인사 평가 피드백을 작성해줘.
이름: 김영호 (입사 2년차, 마케팅팀)
잘한 점: SNS 콘텐츠 기획력 좋음, 팀 분위기 메이커
개선점: 마감 관리가 아쉬움, 데이터 분석 역량 부족
올해 성과: 인스타 팔로워 30% 증가, 캠페인 3건 성공
톤: 긍정적이지만 솔직하게. 개선점은 성장 관점에서 표현.

영업/Sales ​

제안서 초안 ​

text
B사에 보낼 솔루션 제안서를 작성해줘.
고객: B사 (중견 제조업, 직원 500명)
니즈: 사내 ERP 시스템 교체 / 예산: 약 2억원
우리 강점: 클라우드 기반, 제조업 특화 모듈, 24/7 지원
구성: 표지 / 고객 현황 분석 / 솔루션 소개 / 도입 효과 / 일정 / 견적

콜드 이메일 A/B 테스트 ​

text
잠재 고객에게 보낼 콜드 이메일을 써줘.
- 대상: 중소기업 IT 담당자
- 우리 제품: 클라우드 백업 솔루션
- 후킹 포인트: "최근 랜섬웨어 피해 기업 70% 증가"
- 목표: 15분 온라인 미팅 잡기
- 3가지 버전으로 A/B 테스트할 수 있게

마케팅 ​

SNS 콘텐츠 기획 ​

text
이번 주 인스타그램 콘텐츠 5개를 기획해줘.
브랜드: 프리미엄 반려동물 사료
타겟: 25-40세 반려인 / 톤: 전문적이지만 따뜻한
각 콘텐츠별: 캡션(이모지+해시태그) / 이미지 컨셉 / 게시 시간 / CTA

보도자료 ​

text
신제품 출시 보도자료를 작성해줘.
- 제품명: SmartHome Pro 3.0
- 핵심 기능: AI 기반 에너지 절약, 음성 제어
- 출시일: 4월 1일 / 가격: 29만 9천원
- 인용문 포함: CEO 김철수, CTO 이미영
- A4 1.5장 분량, 역삼각형 구조

재무/경영지원 ​

경비 정산 분류 ​

text
"3월_법인카드_내역.csv" 파일을 분석해줘.
1. 사용 항목을 회계 계정별로 분류 (교통비/식대/소모품/접대비/교육비/기타)
2. 부서별 사용 금액 합계
3. 전월 대비 증감
4. 특이 항목 (30만원 이상 건) 리스트
5. 결재 보고용 요약문 작성

고객서비스/CS ​

CS 응대 템플릿 10종 ​

text
우리 제품(구독형 클라우드 서비스) CS 응대 템플릿을 만들어줘.
상황: 결제 실패 / 환불 요청 / 기능 문의 / 서비스 장애 / 요금제 변경
      계정 복구 / 데이터 삭제 / 할인 문의 / 제휴 문의 / 클레임 접수
각 템플릿에 인사말, 본문, 마무리 + [이름],[날짜] 치환 변수 표시

총무/관리 ​

사내 공지 ​

text
사무실 리모델링 관련 전 직원 공지 써줘.
기간: 4/14~4/25 / 범위: 3층 미팅룸, 휴게실
영향: 해당 기간 3층 미팅룸 사용 불가
대안: 2층 대회의실, 4층 소회의실 이용
불편을 최소화한다는 메시지 + 대체 공간 예약 방법 안내

IT / 개발팀 ​

장애 보고서 작성 ​

text
오늘 발생한 서비스 장애에 대한 보고서를 작성해줘.

- 장애 시간: 14:20~15:05 (약 45분)
- 영향 범위: 결제 모듈 전체 (약 1,200건 결제 실패)
- 원인: DB 커넥션 풀 초과
- 대응: 커넥션 풀 사이즈 확대 + 서버 재시작
- 재발 방지: 커넥션 모니터링 알림 추가 예정

구성: 장애 개요 / 타임라인 / 원인 분석 / 대응 내역 / 재발 방지 대책
경영진과 유관 부서에 공유할 수 있는 형태로.

기술 문서 비개발자용 번역 ​

text
이 API 문서를 영업팀이 이해할 수 있게 바꿔줘.

[API 문서 내용 붙여넣기]

- 기술 용어는 쉬운 말로 풀어서 설명
- "이 기능을 쓰면 고객에게 어떤 가치가 있는지" 관점으로
- 영업팀이 고객 미팅에서 설명할 수 있는 수준으로

법무 / 컴플라이언스 ​

계약서 검토 포인트 정리 ​

text
이 계약서에서 우리에게 불리한 조항을 찾아줘.

[계약서 내용 붙여넣기]

특히 확인해줄 것:
- 자동 갱신 / 해지 조건
- 손해배상 범위와 상한
- 지식재산권 귀속
- 비밀유지 의무 기간
- 불합리한 일방적 변경 권한

각 조항별로 "왜 불리한지"와 "수정 제안"을 함께 정리해줘.

사내 정책 문서 작성 ​

text
개인정보 처리 방침을 작성해줘.

우리 서비스는:
- B2B SaaS (고객사 직원 정보 처리)
- 수집 항목: 이름, 이메일, 전화번호, 소속 부서
- 보관 기간: 계약 종료 후 1년
- 제3자 제공: 클라우드 호스팅(AWS)

개인정보보호법 기준으로 필수 항목 빠짐없이 작성해줘.

교육 / 연수팀 ​

교육 커리큘럼 설계 ​

text
신입사원 2주 온보딩 프로그램을 설계해줘.

조건:
- 대상: 올해 상반기 신입 15명 (비개발 직군)
- 기간: 2주 (10일, 하루 8시간)
- 포함할 내용: 회사소개, 부서별 업무, 사내 시스템 교육, 비즈니스 매너, 멘토링
- 마지막 날: 팀별 미니 프로젝트 발표

일자별 시간표 + 각 세션의 목표와 준비물도 함께.

퀴즈 & 평가 문항 제작 ​

text
정보보안 교육 후 평가 문항 20개를 만들어줘.

조건:
- 난이도: 쉬움 5개, 보통 10개, 어려움 5개
- 유형: 객관식(4지선다) 15개, O/X 5개
- 범위: 비밀번호 관리, 피싱 대응, 문서 보안, 외부 반출 금지
- 각 문항에 정답과 해설 포함
- 70점(14개) 이상 합격 기준

R&D / 연구개발 ​

기술 트렌드 리서치 정리 ​

text
2026년 AI 기술 트렌드를 분석하고 우리 제품에 적용 가능한 것을 추천해줘.

우리 제품: 제조업 품질관리 솔루션
관심 분야: 컴퓨터 비전, 이상 탐지, 예측 유지보수

각 트렌드별로:
- 기술 개요 (2-3줄)
- 시장 성숙도 (도입기/성장기/성숙기)
- 우리 제품 적용 시나리오
- 예상 투자 규모 (대/중/소)
- 경쟁사 동향

특허/논문 요약 ​

text
이 논문의 핵심 기술을 비전문가도 이해할 수 있게 요약해줘.

[논문 내용 또는 파일 첨부]

요약 형태:
1. 한 줄 요약 (이 연구가 왜 중요한지)
2. 핵심 기술 설명 (비유 활용)
3. 기존 방식과의 차이점
4. 우리 제품에 적용 가능성
5. 한계점과 추가 연구 필요 사항
