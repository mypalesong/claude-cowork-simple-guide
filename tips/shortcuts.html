Claude Desktop 단축키 ​

기본 단축키 ​

단축키기능설명
Ctrl/Cmd + N새 대화새로운 대화를 시작합니다
Ctrl/Cmd + K빠른 검색이전 대화를 빠르게 찾습니다
Ctrl/Cmd + Shift + N새 프로젝트새 프로젝트를 생성합니다
Ctrl/Cmd + Enter메시지 전송작성 중인 메시지를 전송합니다
Ctrl/Cmd + Shift + C코드 블록 복사코드 블록을 클립보드에 복사합니다
Escape생성 중지응답 생성을 즉시 중단합니다
Ctrl/Cmd + /단축키 도움말사용 가능한 단축키 목록을 표시합니다

Cowork 전용 단축키 ​

단축키기능
Tab자동완성 수락
Ctrl/Cmd + Z마지막 작업 되돌리기
Ctrl/Cmd + D파일 패널 토글

숨은 기능 & 팁 10가지 ​

  1. 슬래시 명령어 — /로 시작하면 빠른 명령어 사용 가능
  2. 드래그 앤 드롭 — 파일을 대화창에 끌어다 놓으면 자동 첨부
  3. 멀티 파일 선택 — Shift 클릭으로 여러 파일 한 번에 선택
  4. 대화 분기 — 이전 메시지에서 다른 방향으로 대화 분기 가능
  5. 핀 고정 — 중요한 대화를 핀으로 고정해서 빠르게 접근
  6. 스크린샷 붙여넣기 — Ctrl+V로 스크린샷 바로 붙여넣기
  7. 마크다운 지원 — 입력에 마크다운 사용 가능 (**, ##, - 등)
  8. 대화 내보내기 — 대화 내용을 파일로 저장 가능
  9. 다크/라이트 모드 — 시스템 설정 따라가거나 수동 전환
  10. 음성 입력 — 마이크 버튼으로 음성으로 입력 가능
